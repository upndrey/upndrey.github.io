Типовой проект ВС-02-10: Сальники для прохода металлических труб Ду50-1200

Назад

Сальники, разработанные в настоящем проекте, предназначены для пропуска металлических труб: стальных по ГОСТ 4015-48, ГОСТ 8732-58; водопроводных по ГОСТ 3262-55 и чугунных по ГОСТ 5525-50 через каменные, бетонные и железобетонные стены всех типов сооружений водопровода и канализации, наполненных или смываемых водой во всех климатических районах и при всех грунтовых условиях.
